What is Heap and Why? Heap is a type of memory used in programs to store data that needs to be dynamically allocated and can change in size during the program’s execution. The programmer can request chunk of memory from the heap and the heap manager will allocate that chunk to the program. To request a chunk of memory on heap we use malloc function and provide the size of memory that we need and it will return a pointer to that allocated chunk and using free function we can deallocate that chunk form the heap. It is very effective method for allocating memory which lives in a program for long period time and the size of the memory can be expended when its fully exhausted.
